# **App Name**: GPA Booster

## Core Features:

- Dynamic Subject Input: Interactive form to dynamically add and remove subject entries, including fields for subject name, marks, credits, and target score.
- GPA Calculation API Integration: Call Python backend API using fetch/axios with user-inputted data and display the returned results, including current GPA, target GPA, and required improvement per subject.
- Prioritized Subjects: Use the Python backend API to get the prioritized list of subjects.
- Performance Charts: Generate visual charts (e.g., bar graphs) to display academic performance insights.
- Real-time UI Updates: Implement loading indicators and real-time UI updates to enhance user experience while waiting for the backend to serve an API response.
- Error Handling: Implement robust error handling to manage invalid inputs, failed API calls, or unexpected backend responses and to clearly communicate these issues to the user.

## Style Guidelines:

- Primary color: Neon blue (#42C2FF) to reflect the futuristic, tech-inspired nature of the tool.
- Background color: Dark black (#121212) to enhance the visibility of neon elements and provide a sleek appearance.
- Accent color: Electric Indigo (#6F00FF), a more saturated and vibrant color analogous to neon blue, will be used for highlights.
- Body and headline font: 'Space Grotesk' (sans-serif) for a computerized and techy feel. 'Inter' (sans-serif) for body when the text is longer.
- Use minimalistic and tech-inspired icons in neon blue to complement the modern SaaS dashboard design.
- Implement a modern SaaS-style dashboard layout with dark glass cards and soft glow effects for a futuristic UI.
- Employ smooth animations and transitions to provide a seamless user experience and responsive interactions.
